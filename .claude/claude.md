```
문제: 플레이어가 플랫폼 옆면에서 위로 올라가는 버그

원인 분석:
- check_platform_collision 함수의 충돌 검사 로직이 방향을 제대로 구분하지 못함
- 수직 충돌과 수평 충돌을 구분해야 함

해결 방법:
1. 충돌 방향 판단 로직 추가
   - 플레이어의 이전 위치를 저장하거나
   - y_velocity를 확인하여 아래로 떨어지는 중인지 판단

2. 조건 개선
   - 아래로 떨어질 때만(y_velocity > 0) 플랫폼 위에 착지
   - 플레이어 하단이 플랫폼 상단 근처에 있을 때만 착지 처리
   - 임계값(threshold)을 사용하여 옆면 충돌 구분

3. 필요한 조건들:
   - player_rect.bottom이 platform_rect.top 근처인가?
   - 중력 방향으로 움직이는 중인가?
   - 이전 프레임에서 플레이어가 플랫폼 위에 있었는가?
```
